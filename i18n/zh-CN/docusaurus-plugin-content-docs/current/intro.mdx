# 介绍

Arthas-UI 是一款 IDEA 插件，它能够让你在 IDEA 直接 Attach 到 JVM 中，并执行相关的命令。

特性:

- 📖: 支持自定义 Ognl 脚本来过滤 JVM，不再需要自己找 PID
- 📦: 自动复制 Jdk 和 Arthas 到容器中
- 🖊: 每条命令的执行结果将会被单独记录，支持清屏，查看历史记录
- 🔎: 支持 Arthas 命令的自动补全以及语法检查 ( *不稳定，仍在改进中* )
- 💡: 支持点击 Arthas 的输出跳转到对应的代码
